# 🌤️ Modern Weather Dashboard

A beautiful and responsive weather application that provides real-time weather updates for any city in the world. This project focuses on a clean **Glassmorphism UI** and seamless API integration.

---

## ✨ Features
* 🔍 **Real-time Search:** Get instant weather data for any city.
* 💎 **Modern UI:** Designed with Glassmorphism effects for a premium look.
* 🌡️ **Detailed Info:** Shows temperature, humidity, and wind speed.
* 📱 **Responsive Design:** Works perfectly on both desktop and mobile screens.
* ☁️ **Dynamic Icons:** Weather icons change automatically based on the current weather condition (Rain, Clouds, Clear, etc.).

---

## 🛠️ Tech Stack
* **HTML5:** Structured the application layout.
* **CSS3:** Styled with modern layouts, gradients, and blur effects.
* **JavaScript (ES6+):** Handled API requests and DOM manipulation.
* **OpenWeatherMap API:** Used to fetch live weather data.
